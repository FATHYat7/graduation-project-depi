# 🛒 E-Commerce Web Application

A full-featured E-Commerce web application built using **React.js** and **Vite**, providing a complete and responsive online shopping experience.

---

## 🚀 Features
- 🛍️ Browse products with dynamic data fetching  
- 🛒 Add / remove items from cart  
- ❤️ Wishlist functionality  
- 🔐 User authentication (Login / Register)  
- 🔎 Search and filter products  
- 💳 Checkout & simulated payment flow  
- 📱 Fully responsive UI  
- 🎞️ Smooth animations and interactive sliders  

---

## 🛠️ Tech Stack
- React.js  
- Vite  
- Axios  
- React Router DOM  
- Bootstrap / React Bootstrap  
- Swiper  
- CSS
